# Smooth Flight - PRD (Product Requirements Document)

## 1. 개요

### 1.1 프로젝트명
Smooth Flight (가칭)

### 1.2 목표
Google Play의 "Flight Master" (com.WalkTalk.FlightMaster) 앱을 참고하여, 렉 없이 부드럽게 작동하는 항공 관제 아케이드 게임 개발

### 1.3 핵심 차별점
- 기존 앱의 심한 렉/끊김 현상 해결
- 60fps 안정적 유지
- 저사양 기기에서도 원활한 플레이

---

## 2. 게임 개요

| 항목 | 사양 |
|------|------|
| 장르 | 캐주얼 아케이드 - 항공 교통 관제 |
| 플랫폼 | 웹 브라우저 (PC + 모바일 동등 지원) |
| 조작 방식 | 경로 드로잉 (Flight Control 스타일) |
| 그래픽 | 2D 미니멀 벡터 |
| 수익화 | 완전 무료 |

---

## 3. 기술 스택

| 구분 | 기술 |
|------|------|
| 렌더링 | HTML5 Canvas + Vanilla JavaScript |
| 데이터 저장 | LocalStorage |
| 사운드 | Web Audio API |
| 입력 | 마우스 + 터치 동시 지원 |

---

## 4. 핵심 게임플레이

### 4.1 기본 메커니즘
1. 화면 가장자리에서 비행기가 등장
2. 플레이어가 비행기를 탭/클릭하여 선택
3. 드래그로 경로를 그려서 활주로까지 유도
4. 비행기가 경로를 따라 이동하여 착륙
5. 비행기 간 충돌 시 게임오버

### 4.2 난이도 요소
- 시간이 지날수록 비행기 등장 빈도 증가
- 비행기 속도 다양화 (소형기 빠름, 대형기 느림)
- 활주로 개수: 레벨별 2~4개

### 4.3 튜토리얼
- 없음 (직관적 UI로 자연스럽게 학습)

---

## 5. 핵심 기능

### 5.1 다양한 공항/레벨
- 최소 5개 이상의 공항 스테이지
- 난이도별 차별화된 레이아웃
- 활주로 배치, 장애물 등 변화

### 5.2 비행기 종류
| 종류 | 속도 | 크기 | 점수 |
|------|------|------|------|
| 소형기 | 빠름 | 작음 | 낮음 |
| 대형기 | 느림 | 큼 | 높음 |
| 헬기 | 중간 | 작음 | 중간 |

### 5.3 시간/날씨 시스템
- 시간: 낮 / 밤
- 날씨: 맑음 / 비 / 안개
- 시각적 효과 + 난이도 영향

### 5.4 점수/랭킹 시스템
- 착륙 성공 시 점수 획득
- 연속 착륙 콤보 보너스
- LocalStorage 기반 로컬 랭킹

### 5.5 사운드
- 배경음악 (BGM)
- 효과음: 착륙, 충돌, UI 인터랙션

---

## 6. 성능 최적화 전략

### 6.1 목표
- 60fps 안정적 유지
- 모바일 저사양 기기 지원
- 메모리 누수 방지

### 6.2 최적화 기법
| 기법 | 설명 |
|------|------|
| 오브젝트 풀링 | 비행기/이펙트 재사용으로 GC 최소화 |
| requestAnimationFrame | 브라우저 최적화된 렌더 루프 |
| 더티 렉탱글 | 변경된 영역만 다시 그리기 |
| 효율적 충돌 감지 | 공간 분할 또는 거리 기반 최적화 |
| 이미지 스프라이트 | 네트워크 요청 최소화 |

---

## 7. UI/UX

### 7.1 화면 구성
- 메인 메뉴: 플레이, 레벨 선택, 설정, 랭킹
- 게임 화면: 공항 맵, 점수 표시, 일시정지 버튼
- 게임오버: 점수, 재시작, 메뉴로

### 7.2 반응형 디자인
- PC: 마우스 드래그
- 모바일: 터치 드래그
- 화면 크기에 맞춰 자동 스케일링

## 9. 개발 우선순위

### Phase 1: 코어 게임플레이 (MVP)
1. 게임 루프 및 렌더링 시스템
2. 비행기 스폰 및 이동
3. 경로 드로잉 시스템
4. 충돌 감지
5. 기본 점수 시스템

### Phase 2: 콘텐츠 확장
1. 다양한 비행기 종류
2. 여러 공항 레벨
3. 시간/날씨 시스템

### Phase 3: 폴리싱
1. 사운드 시스템
2. UI/UX 개선
3. 성능 최적화
4. 랭킹 시스템

---

## 10. 성공 지표

- 60fps 유지율: 95% 이상
- 첫 로딩 시간: 3초 이내
- 게임플레이 중 프레임 드롭: 없음
